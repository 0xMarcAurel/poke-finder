# PokéFinder
A lightweight Node.js web app that lets users search any Pokémon by name and instantly see its details and official artwork.

App built with Express and EJS, using data from the public [PokéAPI](https://pokeapi.co/).

## Demo
![PokéFinder screenshot](./public/images/app-screenshot.jpg)

## Features
* Search any Pokémon by name
* Shows name, type, height and weight, and official artwork
* Simple, responsive UI
* Built with Express, Axios, and EJS

## Tech stack
### Backend:
* Node.js
* Express
* Axios
### Frontend:
* EJS
* Vanilla CSS
### API:
* [PokéAPI](https://pokeapi.co/)

## Installation
### 1. Clone the repository
`git clone https://github.com/your-username/poke-finder.git`
### 2. Install dependencies
`npm install`
### 3. Run the server
`node src/index.js`
### 4. Visit the app
`http://localhost:3000`
